Animation property

The CSS animation property take a keyframe ( the animation itself ) and its details.

The keyframe is the animation structure:
FROM what/where/how the component should start
TO what/where/how it should end

Animated Backgrounds

Simple animation using images for the background.

Most CSS properties used are: margins, transforms (rotate, in this case), animation-delay, animation-duration

.bg{ position: relative; background-color: #000; width: 100%; height: 100%; overflow-y: hidden; } .logo-ball{ width: 20px; height: 20px; border-radius: 50%; position: absolute; bottom: -20px; animation: levitate, rotate, shine; animation-iteration-count: infinite; } .logo-ball:nth-of-type(1){ animation-duration: 10s, 10s, 3s; animation-delay: 2s, 0s, 0s; } .logo-ball:nth-of-type(2){ animation-duration: 4s, 5s, 3s; animation-delay: 5s, 0s, 0s; } .logo-ball:nth-of-type(3){ animation-duration: 8s, 5s, 3s; animation-delay: 1s, 0s, 0s; } .logo-ball:nth-of-type(4){ animation-duration: 12s, 10s, 3s; animation-delay: 0s; }
